calcularPrecio(numPags, tipoEncuad, cantidad) → {number|undefined}

Calculates the total price of an album order based on the number of pages, binding type, and quantity. - If the number of pages is not even, is greater than 100, or the quantity is greater than 250, the function returns `undefined`. - If the quantity is 10 or more, a 25% discount is applied. - If the quantity is 5 or more, a 10% discount is applied.
Parameters:
Name Type Description
numPags number Total number of pages (must be an even number).
tipoEncuad string Type of binding ('R' for softcover, any other value for hardcover).
cantidad number Number of albums ordered (maximum 250).
